# Live Preview Instructions

To preview the MediStudio pharmacy ecommerce experience locally:

1. Install dependencies if you have not already:
   ```bash
   npm install
   ```
2. Start the Vite development server:
   ```bash
   npm run dev -- --host 0.0.0.0 --port 4173
   ```
3. Open the provided network URL in your browser (for example, `http://localhost:4173/`).

The UI has been verified to load successfully with the Apple-inspired styling requested. See the accompanying pull request for a current screenshot.
